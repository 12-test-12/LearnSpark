# Vue 3 组合式 API 学习笔记

## 核心概念

组合式 API（Composition API）是 Vue 3 引入的新编程模型，通过 `setup` 函数组织组件逻辑。

### ref 与 reactive

- `ref` 用于基本类型，需要通过 `.value` 访问
- `reactive` 用于对象，直接访问属性

### computed 计算属性

计算属性会自动追踪依赖并缓存结果，只有依赖变化时才重新计算。

## 示例代码

```vue
<script setup>
import { ref, computed } from 'vue'
const count = ref(0)
const double = computed(() => count.value * 2)
</script>
```
